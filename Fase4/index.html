Pròposito del Sitio

Este sitio web fue creado con el fin de dar a conocer la estructura básica que una página debe tener al momento de su creación, en ella también se menciona el significado y uso de algunas de las etiquetas que se utilizan normalmente para diseñar páginas con el lenguaje en HTML, además de los atributos y valores que estas deben adquirir para poder funcionar correctamente y de esta manera lograr conseguir conocimientos básicos que se necesitan a la hora de diseñar páginas web.

Introducciòn

El lenguaje HTML (Lenguaje de Marcas de Hipertexto, del inglés HyperText Markup Language) es uno de los lenguajes de programación más fáciles de manejar y entender, ya que nos permite presentar textos de una forma más agradable, de manera que podemos crear vínculos con otros sitios que también hacen referencia al tema visto permitiéndonos de este modo insertar imágenes, audios, videos, gráficos, incluso otros documentos relacionados.

Este leguaje requiere de una estructura básica para así poder iniciar con la creación de la página web, aparte de la estructura básica requerida se necesitan diferentes etiquetas para empezar a programar nuestra página. De esta manera a continuación, se mencionan algunas etiquetas junto con sus tributos y valores.
